Interactive Data Visualizations with Bokeh

What is Bokeh?

  • Data visualization library
  • Built for the web
  • Strong ties to data science

Aside : Imperative vs Declarative

  • Imperative interface: the "how"
    • e.g. d3.js, matplotlib
  • Declarative interface: the "what"
    • e.g. seaborn, ggplot, altair

Bokeh has both an imperative and a declarative interface.

Bokeh Interface

Bokeh Python Code

from bokeh.plotting import figure, show, output_file
from bokeh.sampledata.iris import flowers

colormap = {'setosa': 'red',
            'versicolor': 'green',
            'virginica': 'blue'}
colors = [colormap[x] for x in flowers['species']]

p = figure(title = "Iris Morphology")
p.xaxis.axis_label = 'Petal Length'
p.yaxis.axis_label = 'Petal Width'

p.circle(flowers["petal_length"], flowers["petal_width"],
         color=colors, fill_alpha=0.2, size=10)

output_file("iris.html", title="iris.py example")

show(p)

BokehJS Models and Views

  • Based on Backbone.js
  • Backbone Models store the state (data)
  • Backbone Views know how to render the state (canvas API)
  • Views listen to events and render on change
